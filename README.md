# Spicy Lyrics Nobg

A fork of [Spicy Lyrics](https://github.com/Spikerko/spicy-lyrics) that adds an option to disable the dynamic background behind the lyrics view, reducing RAM usage.

## Installation

1. Make sure you have [Spicetify](https://spicetify.app) installed
2. If you have Spicy Lyrics installed from the Marketplace, uninstall it first
3. Download `spicy-lyrics.js` from the [latest release](https://github.com/Oarner/spicy-lyrics-nobg/releases/latest)
4. Put the file inside your Spicetify Extensions directory (`%appdata%\spicetify\Extensions` on Windows)
5. Run `spicetify config extensions spicy-lyrics.js`
6. Run `spicetify apply`
7. All done! Find the toggle under Spicy Lyrics settings → Background → Show Lyrics Background
